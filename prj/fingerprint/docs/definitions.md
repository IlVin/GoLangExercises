# Определения

## Сервис
Сервис, предоставляемый по интернет протоколам, в большинстве случаев HTTP протокол

## Пользователь
Что-то или кто-то с интеллектом, воспользовавшийся по интернет протоколам **сервисом**.
Пользователь может быть аутентифицирован Сервисом.

## Устройство пользователя
Это связка физического устройства пользователя и установленной на устройстве программы, осуществляющей использование **Сервиса** по интернет протоколам.
У Пользователя может быть несколько устройств и на каждом устройстве может быть установлено несолько программ.

## Цифровой отпечаток устройства
**DFP** - Digital FingerPrint
Это токен, который позволяет аутентифицировать устройство **Пользователя**, т.е. установить факт, что устройство принадлежит Пользователю.

## Первичный цифровой отпечаток
Это цифровой отпечаток **устройства**, с которым пользователь произвел первый контакт с **Cервисом**.
У **Пользователя** может быть несколько устройств, значит может быть несколько первичных цифровых отпечатков.

## Эталонный цифровой отпечаток
Это цифровой отпечаток устройства пользователя, с которым пользователь провел финансовую операцию.
Так как финансовая операция требует аутентификации и авторизации пользователя, то можно считать, что эталонный отпечаток - это DFP, с которым пользователь был аутентифицирован.
У **Пользователя** может быть несколько устройств, значит может быть несколько эталонных цифровых отпечатков.
Так как устройства пользователя и их настройки могут меняться со временем, то эталонный цифровой отпечаток устройства может быть обновлен. Старый эталонный отпечаток устройства становится архивным цифровым отпечатком.

## Архивный цифровой отпечаток
Это эталонный цифровой отпечаток, который был заменен новым эталонным DFP.
ЦБ требует хранить архивные DFP не менее 5 лет.

## Время жизни цифрового отпечатка устройства
**Первичный цифровой отпечаток** - живет вечно.
**Эталонный цифровой отпечаток** - Время жизни задается **Сервисом**. По умолчанию 30 дней. Через TTL секунд эталонный DFP становится архивным DFP
**Архивный цифровой отпечаток** - Время хранения 5+ лет. Требование ЦБ.

## Время активности цифрового отпечатка устройства
Время активности DFP - это время с момента регистраци DFP в качестве эталонного, в течение которого DFP может использоваться для аутентификации устройства пользователя.
