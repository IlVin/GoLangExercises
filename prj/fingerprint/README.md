# DFP - цифровой отпечаток устройств

## Введение

В 2023 году ЦБ РФ выпустил [стандарт безопасности банковских операций](https://cbr.ru/Crosscut/LawActs/File/6177), в котором описал технологию цифрового отпечатка устройства.

В этом учебном проекте я буду реализовывать микросервис на ЯП GoLang регистрации и валидации эталонных цифровых отпечатков устройств.
